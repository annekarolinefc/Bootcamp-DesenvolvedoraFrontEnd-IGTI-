É possivel enfatizar uma palavra com o elemento em, dar ainda mais ênfase com o elemento strong, ou marcar uma palavra com o elemento mark Embora existam os elementos i(itálico) e b () negrito, eles nao são mais recomendados, pois o HTML deve focar no conteúdo e semnântica, deixando a estilização para o CSS.

Também podeos demarcar código fonte com o elemento code, ou demarcar entrada de teclado com o elemento kbd.

Podemor forçar uma quebra de linha aqui com o elemento br.


Podemos usar sub para subsrito, como H 2O, ou sup para sobrescrito, como em x 2

Imagens são inseridar com o elemento img.

A imagem pode ser hospedada em outro site.

O elemento a corresponde a um link para outro documento
